# Good Hosts

主要解决部分网址存在`DNS`污染加速网站访问

默认配置了：`github`、`dribbble`

## 特性

- 支持多`DoH`同时测速
- 支持择优选择延迟低的`IP`
- 支持自定义配置`DoH`地址
- 支持自定义配置加速网址

## 使用方法

### 1. 自动更新

建议搭配：[SwitchHosts](https://swh.app/) 使用

更新地址：

自动更新时间调整为1小时

### 2. 添加到 hosts

将 [hosts](https://github.com/ifui/GoodHosts/blob/main/hosts) 中的内容添加到你的`hosts`文件中
